━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

D3MAND est un site communautaire pour soutenir la création de Destiny 3.
Stack : React 19 + TypeScript + Vite 7 + Tailwind CSS 4.
100 % statique côté client — aucun back-end propre.
Déploiement géré par Manus sur un serveur externe.
Deux agents IA collaborent : Claude (toi) pour le code, Manus pour le déploiement.

Hook useHubStats (client/src/hooks/useHubStats.ts) :
  - Lit le même CSV Google Sheet que les pétitions
  - Lignes spéciales : id="hours" et id="cost", valeur en colonne C
  - Valeurs par défaut : 26h et 120€ (affichées même sans Sheet configuré)
  - Toujours affiché (pas conditionnel sur null)

Pour mettre à jour les compteurs :
  Ouvrir la Google Sheet "D3MAND Petitions"
  → modifier la valeur en colonne C des lignes hours et cost
